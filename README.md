# Exploratory Data Analysis (EDA) on E-Commerce Dataset

## Project Overview
This project performs Exploratory Data Analysis (EDA) on an E-Commerce dataset using Python. The analysis helps identify sales trends, customer behavior, payment preferences, revenue patterns, and outliers through statistical methods and data visualization.

## Objectives
- Understand the structure of the dataset
- Calculate statistical measures such as mean, median, and count
- Analyze product sales and revenue
- Study order status distribution
- Examine payment methods used by customers
- Detect outliers in the dataset
- Visualize trends using charts and graphs

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Dataset Features
The dataset contains the following information:
- Order ID
- Order Date
- Customer ID
- Product
- Quantity
- Unit Price
- Payment Method
- Order Status
- Referral Source
- Total Price

## Analysis Performed
1. Data Loading and Inspection
2. Missing Value Analysis
3. Descriptive Statistics
4. Product-wise Sales Analysis
5. Revenue Analysis
6. Payment Method Analysis
7. Order Status Distribution
8. Outlier Detection using Box Plot
9. Monthly Sales Trend Analysis
10. Correlation Analysis using Heatmap

## Visualizations
- Product Sales Bar Chart
- Order Status Pie Chart
- Payment Method Count Plot
- Referral Source Analysis
- Quantity Distribution Histogram
- Total Price Distribution Histogram
- Outlier Detection Box Plot
- Monthly Sales Trend Line Chart
- Correlation Heatmap
- Top Revenue Generating Products Chart

## How to Run the Project

### Install Required Libraries

```bash
pip install pandas numpy matplotlib seaborn
```

### Run the Project

```bash
python project2_EDA.py
```

## Key Findings
